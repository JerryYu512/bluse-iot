# 存放枚举定义

```shell
├── e_basic.proto       # 基础参数枚举
├── e_common.proto      # 通用枚举，与配置参数无关
├── e_custom.proto      # 自定义枚举，开发者需要借用该实现方式
├── e_db.proto          # 数据库配置枚举
├── e_device.proto      # 设备配置参数枚举
├── e_hw.proto          # 硬件配置枚举
├── e_media.proto       # 媒体配置枚举
├── e_module.proto      # 模块配置枚举
├── e_network.proto     # 网络配置枚举
├── e_product.proto     # 产品配置枚举
├── e_storage.proto     # 存储配置枚举
├── e_subsys.proto      # 子系统配置枚举
└── e_system.proto      # 系统配置枚举

```
